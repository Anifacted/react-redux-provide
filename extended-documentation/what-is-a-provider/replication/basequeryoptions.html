baseQueryOptions

baseQueryOptions

Optional object that can be used as the base of each query's options. For example, if you want to limit the query results to some number by default:

const replication = {
  reducerKeys: [
    'userId',
    'userName',
    'userPasswordHash'
  ],
  queryable: [
    'userName',
    'userDeleted'
  ],
  baseQuery: {
    userDeleted: false
  },
  baseQueryOptions: {
    limit: 100
  }
};
